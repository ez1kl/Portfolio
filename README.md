# Portfolio Personnel

Portfolio web moderne développé avec React, Three.js et TailwindCSS.

## Technologies

- React 18
- Three.js
- Framer Motion
- TailwindCSS
- Vite

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Le site sera accessible sur `http://localhost:5173/`

## Build

```bash
npm run build
```
